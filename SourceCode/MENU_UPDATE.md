## 🚀 使用方式

### 1. 启动应用

```bash
# 进入 SourceCode 目录
cd SourceCode

# 启动开发服务器（需要配置构建工具）
npm run dev
```

### 2. 访问页面

- **首页**: http://localhost:3000/home
- **源代码**: http://localhost:3000/form-import
- **帮助**: http://localhost:3000/help

### 3. 菜单导航

点击顶部导航栏的菜单项即可在不同页面间切换：
- 点击 **首页** 查看系统概览和快速操作
- 点击 **源代码** 进入表单导入功能
- 点击 **帮助** 查看使用说明和常见问题

## 🔧 技术实现

### 1. 组件架构
```
App.vue (根组件)
└── Layout.vue (布局组件)
    ├── 导航栏
    ├── 路由视图
    │   ├── Home (首页)
    │   ├── FormImport (源代码)
    │   └── Help (帮助)
    └── 页脚
```

### 2. 路由配置
```typescript
const routes = [
  { path: '/', redirect: '/home' },
  { path: '/home', component: Home },
  { path: '/form-import', component: FormImport },
  { path: '/help', component: Help },
];
```

### 3. 状态管理
- 使用 Vue 3 Composition API
- 响应式数据绑定
- 组件间通信通过 props 和 emits
