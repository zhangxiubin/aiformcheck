## ⚙️ 配置说明

### 开发服务器配置
```typescript
// vite.config.ts
{
  server: {
    port: 3000,        // 端口号
    open: true,        // 自动打开浏览器
    cors: true,        // 启用CORS
  }
}
```

### 路由配置
```typescript
// router/form-import.routes.ts
const routes = [
  { path: '/', redirect: '/home' },
  { path: '/home', component: Home },
  { path: '/form-import', component: FormImport },
  { path: '/help', component: Help },
];
```

## 🐛 故障排除

### 常见问题

#### 1. 端口被占用
```bash
# 错误信息：Port 3000 is in use
# 解决方法：修改 vite.config.ts 中的端口号
server: {
  port: 3001,  // 改为其他端口
}
```

#### 2. 依赖安装失败
```bash
# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

#### 3. 页面空白
```bash
# 检查浏览器控制台是否有错误
# 确保所有依赖都已正确安装
npm run dev
```
